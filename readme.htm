Jarvis - Desktop AI Assistant

Jarvis is a voice-based AI assistant that operates in three distinct modes: Chat Mode, Command Mode, and Code Mode. Designed to help with both general and technical tasks, Jarvis can chat like other large language models (LLMs) and perform a wide range of desktop and internet-based functions.

Features

Chat Mode

In Chat Mode, Jarvis can answer queries and provide useful information like any other LLM, while also providing the following functional capabilities:

  • General Functionalities:
    • get_ip_address: Retrieve the IP address of the device.
    • get_time: Check the current time.
    • set_reminder: Set a reminder.
    • get_temp: Retrieve the current temperature.
    • exit: Exit Chat Mode.
    • get_date: Check today’s date.

Command Mode

In Command Mode, Jarvis can perform various desktop and web-based operations. Command Mode functionalities are divided into commands that require arguments and those that don’t.

  • Commands Requiring Arguments:
    • open_app: Open a specified application.
    • close_app: Close a specified application.
    • search_on_youtube: Search for a specified topic on YouTube.
    • search_on_internet: Perform an internet search for a specified query.
    • generate_image: Generate an image based on a given prompt.
  • Commands Without Arguments:
    • YouTube Controls:
      • open_youtube: Open YouTube.
      • pause_video: Pause the current YouTube video.
      • restart_video: Restart the current video.
      • mute_video: Mute the current video.
      • skip_video: Skip to the next video.
      • back_video: Go back to the previous video.
      • fullscreen_video: Enter fullscreen mode for the video.
      • film_mode: Enter film mode on YouTube.
      • close_youtube: Close YouTube.
    • Browser Controls:
      • close_tab: Close the current browser tab.
      • open_browser: Open the default web browser.
      • open_recently_closed_tab: Reopen the last closed tab.
      • open_download: Open the downloads page in the browser.
      • open_history: Open the browsing history.
      • open_new_window: Open a new browser window.
      • open_new_tab: Open a new tab in the browser.
    • System Controls:
      • take_screenshot: Take a screenshot of the screen.
      • shutdown_sys: Shut down the system.
      • restart_sys: Restart the system.
    • Exit:
      • exit: Exit Command Mode.

Code Mode

In Code Mode, Jarvis can assist with basic programming tasks. Like Command Mode, the functionalities in Code Mode are categorized based on whether arguments are required. All the operations in code mode will take place in ‘vsCode’.

  • Commands Requiring Arguments:
    • generate_code: Generate code based on a specified prompt or requirement.
  • Commands Without Arguments:
    • new_file: Create a new code file.
    • new_project: Start a new project.
    • save_file: Save the current file.
    • close_file: Close the current file.
    • run_code: Execute the current code file.
    • exit: Exit Code Mode.
